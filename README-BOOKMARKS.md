## 快速开始

### 安装

#### 方式一：下载已打包的扩展（推荐）

1. 访问项目 Releases 页面：[github.com/huanyu-a/FavsHub/releases](https://github.com/huanyu-a/FavsHub/releases)
2. 下载最新版本的 `FavsHub-vX.X.X.zip`
3. 解压到本地目录
4. 打开 `chrome://extensions/`（Edge 用户：`edge://extensions/`）
5. 启用右上角「开发者模式」
6. 点击「加载已解压的扩展程序」，选择解压后的文件夹
7. 打开新标签页（`Ctrl + T`），即可看到 FavsHub

#### 方式二：从源码加载

1. 克隆项目：`git clone https://github.com/huanyu-a/FavsHub.git`
2. 打开 `chrome://extensions/`（Edge 用户：`edge://extensions/`）
3. 启用右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择项目根目录
5. 打开新标签页（`Ctrl + T`），即可看到 FavsHub

> 📦 项目地址：[github.com/huanyu-a/FavsHub](https://github.com/huanyu-a/FavsHub)

**无需迁移数据** — FavsHub 直接使用 Chrome 原生书签，所有操作实时双向同步。

### 浏览与打开书签

FavsHub 以 **卡片网格** 形式展示书签，每个书签自动生成独特的渐变色背景：

- **单击书签卡片** → 在新标签页打开该网站（可在设置中改为当前页打开）
- **滚动浏览** → 所有书签按文件夹分组，向下滚动即可浏览全部
- **卡片颜色** → 从网站图标自动提取，每个书签独一无二

> 💡 **提示**：如果某个书签没有颜色，刷新页面即可自动生成。

### 文件夹导航

**三种方式找到目标文件夹：**

**① 侧边栏点击**
- 点击左侧文件夹名 → 主区域自动滚动到对应分组
- 展开/折叠带箭头的子文件夹

**② 主区域文件夹卡片**
- 灰色文件夹卡片 → 点击进入该文件夹的详细视图

**③ 设为默认首页**
- 右键任一文件夹 → 「设为默认首页」→ 每次打开新标签页自动显示该文件夹内容

### 切换主题

点击侧边栏底部第一个图标（🌙/☀️）切换浅色/深色主题，支持跟随系统自动切换。

## 日常使用

### 搜索书签

在搜索框输入关键词，系统自动显示 **多维建议**：

```
🔍 搜索: "github"
━━━━━━━━━━━━━━━━━━━━━━━━
📑 书签匹配          🕐 历史匹配
┌─────────────────────────┐
│ 🔖 GitHub - github.com │
│ 🔖 GitHub Docs          │
│ 🕐 github.com/issues    │
└─────────────────────────┘
  ↑ 键盘 ↑↓ 选择，Enter 打开，所有建议按相关性排序
```

- 书签建议和历史建议可分别在设置中开关
- 匹配优先级：标题完全匹配 > 前缀匹配 > 包含匹配 > 模糊匹配

### 多引擎对比搜索

**一键在多个搜索引擎中同时搜索同一关键词：**

1. 在搜索框输入关键词
2. 按 **`Ctrl + Enter`**（Mac：`Cmd + Enter`）
3. 系统自动在所有启用的搜索引擎中打开结果页

```
输入 "最佳 React 教程" → Ctrl + Enter
    ├── Google 结果
    ├── Bing 结果
    ├── 百度结果
    └── ...（更多引擎）
```

**切换默认引擎**：点击搜索框左侧的搜索引擎图标，弹出引擎选择面板，支持：
- 🤖 AI 搜索（Kimi、豆包、DeepSeek、ChatGPT 等）
- 🌐 通用搜索（Google、Bing、百度、360 等）
- 💬 社交媒体（知乎、B站、GitHub 等）
- ➕ 自定义搜索引擎

### 编辑与修改分类

**编辑书签**：右键任一书签 → 「编辑」，或直接双击书签卡片，弹出编辑对话框：

```
┌─────────────────────────────────┐
│  编辑书签                     ✕ │
│                                 │
│  名称: [GitHub 主页          ]  │
│  URL:  [https://github.com  ]  │
│  分类: [📁 工作          ▾   ]  │
│                                 │
│       [取消]      [保存]        │
└─────────────────────────────────┘
```

- **修改分类**：点击分类下拉框，以树形层级显示所有文件夹，选择目标文件夹即可将书签移入
- 保存后自动调用 Chrome 书签 API 完成移动和更新

### 底部快捷工具

侧边栏底部的工具栏提供一键访问 Chrome 内部页面：

| 图标 | 功能 |
|------|------|
| 🕐 | 浏览历史记录 (`chrome://history`) |
| ⬇ | 下载管理 (`chrome://downloads`) |
| 🔑 | 密码管理 (`chrome://settings/passwords`) |
| 🧩 | 扩展管理 (`chrome://extensions`) |
| ⚙️ | 打开 FavsHub 设置面板 |

## 进阶功能

### 拖拽排序

**书签卡片排序**：按住书签卡片直接拖拽到目标位置，松手后自动同步到 Chrome 书签。

**文件夹树排序**：在左侧导航目录中，文件夹支持：
- **同层排序**：拖拽调整文件夹显示顺序
- **更改层级**：将文件夹拖入另一个文件夹内部（嵌套），或拖出恢复为顶级

拖拽时半透明预览显示，松手后自动持久化。

### 右键菜单

**书签卡片右键：**

| 菜单项 | 说明 |
|--------|------|
| 在新标签页/窗口打开 | 控制打开方式 |
| 在无痕模式打开 | 隐私浏览 |
| 编辑 | 修改名称、URL、分类 |
| 删除 | 确认后删除 |
| 复制链接 | 一键复制 URL |
| 生成二维码 | 手机扫码打开 |

**文件夹右键：**

| 菜单项 | 说明 |
|--------|------|
| 打开所有书签 | 批量打开文件夹内所有书签 |
| 重命名 / 删除 | 管理文件夹 |
| 设为默认首页 | 新标签页默认显示该文件夹 |
| 取消默认首页 | — |

### 悬浮球快捷访问

开启后，**任意网页**右上角出现浮动图标，无需离开当前页面即可：

- 🔍 **快捷搜索**：点击悬浮球展开面板，直接搜索
- 📝 **选中文字搜索**：在网页选中文字 → 点击悬浮球 → 选择搜索引擎 → 直接跳转搜索结果

> 在设置 → 悬浮球设置中可开关此功能。

### 侧边栏模式

在浏览器右侧打开紧凑的书签面板，适合边浏览边查书签：

- 快捷键：`Alt + B`（Mac：`Cmd + B`）
- 侧边栏内可直接搜索、浏览书签，点击书签后可配置在新标签页或侧边栏内打开

## 常见问题

### 书签数据在哪里？会丢失吗？

所有数据存储在 **Chrome 原生书签系统**中，FavsHub 不创建独立数据库。操作实时双向同步，卸载 FavsHub 不会丢失书签。

### 书签卡片的颜色如何生成？

从网站的 favicon 自动提取主色和次色，应用为渐变背景。首次加载后缓存在本地，后续秒开。

### 如何备份书签？

Chrome 书签管理器（`chrome://bookmarks`）→ 导出 HTML 文件即可。

### 拖拽后顺序恢复了？

拖拽立即调用 Chrome API 更新，系统每 30 秒同步显示。如顺序恢复，刷新页面即可。

### 如何添加自定义搜索引擎？

设置 → 搜索设置 → 搜索引擎管理 → 填写名称和 URL 模板（`{query}` 代表搜索词）。
