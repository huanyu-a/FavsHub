## 快速开始

### 安装 FavsHub

1. **加载扩展**
   - 打开 `chrome://extensions/`（Edge 用户打开 `edge://extensions/`）
   - 右上角启用"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 FavsHub 项目根目录

2. **开始使用**
   - 打开新标签页（`Ctrl + T` 或 `Cmd + T`）
   - 自动进入 FavsHub 书签导航页面
   - 您的 Chrome 书签已自动显示

## 功能详解

### 🎨 智能彩色书签卡片

**自动渐变色背景，让每个书签都独一无二**

#### 功能特色
- 🎨 **自动生成颜色**：系统根据网站图标（favicon）自动提取主色和次色，生成独特的渐变色背景
- 💾 **智能缓存**：首次加载后缓存颜色，后续访问秒级显示
- ✨ **悬停动效**：鼠标悬停时卡片微微上浮，阴影加深

#### 使用技巧
- 书签卡片**点击即可打开**对应网站
- 卡片右上角显示网站 favicon 和名称
- 如果某些书签没有颜色，刷新页面即可解决

### 📁 文件夹导航

**三位一体的导航系统，快速定位目标书签**

#### 三种导航方式

##### 1. 左侧树形导航栏
```
📁 书签栏
  ├ 📂 工具

  │   └ 📂 设计工具

**操作方法：**

### 🔍 多引擎对比搜索

**20+ 搜索引擎，对比搜索助您找到最佳答案**

#### 对比搜索（🔥 核心亮点）

**一键在所有搜索引擎中搜索同一内容，对比不同结果**

##### 方式 1：新标签页对比搜索

**操作步骤：**
1. 打开新标签页（`Ctrl/Command + T`）
2. 在搜索框输入关键词
3. 按下 **`Cmd/Ctrl + Enter`**
4. ✨ **系统自动在所有启用的搜索引擎中同时搜索！**
5. 所有搜索结果在新标签页中打开，并自动分组

```
🔍 搜索框输入: "Python 教程"
按下 Ctrl + Enter
        ↓
┌─────────┐ ┌─────────┐ ┌─────────┐
│ Google  │ │  Bing   │ │  百度   │
│ 结果页  │ │ 结果页  │ │ 结果页  │
└─────────┘ └─────────┘ └─────────┘
    ↑           ↑           ↑
  同时打开，自动对比，找到最佳答案！
```

**对比搜索优势：**
- ✅ **一次搜索，多引擎对比**：无需反复复制粘贴
- ✅ **自动分组**：搜索结果标签页自动归为一组
- ✅ **高效全面**：同时查看不同引擎的结果，不遗漏信息
- ✅ **智能识别**：自动识别关键词，精准搜索

##### 方式 2：任意网页选中文本搜索

**在任意网页中选中文字后，高效启动搜索：**

| 触发方式 | 操作步骤 | 适用场景 |
|---------|---------|---------|
| **悬浮球搜索** | 选中文字 → 点击悬浮球 → 点击搜索引擎 | 浏览网页时快速搜索 |
| **侧边栏快捷键** | 选中文字 → `Alt + B` → `Alt + 1-7` | 需要精确控制搜索引擎 |
| **全部搜索** | 选中文字 → `Ctrl/Command + Enter` | 需要对比所有引擎结果 |

**支持的搜索引擎快捷键：**
```
Alt + 1 → Google    Alt + 2 → Bing      Alt + 3 → 百度
Alt + 4 → 360搜索   Alt + 5 → ChatGPT   Alt + 6 → Kimi
Alt + 7 → 豆包      ...（依此类推）
```

##### 对比搜索使用场景

**场景 1：技术调研**
```
搜索 "React vs Vue 性能对比"
→ Ctrl + Enter → 同时打开 Google、Bing、百度

```

**场景 2：学习新概念**

```

**场景 3：购物比价**
```
搜索 "iPhone 15 Pro 评测"
→ Ctrl + Enter → 多引擎同时搜索
→ 对比不同来源的评测 → 全面了解产品优劣
```

### 🔮 悬浮球快捷访问

**任意网页上的快速访问中心**

##### 什么是悬浮球？

悬浮球是一个固定在**任意网页**右上方的浮动图标，让您在浏览任何网页时，都能快速访问搜索引擎和书签，无需切换标签页：

```
┌──────────────────────────────────────────┐
│  您正在浏览的网页内容...                   │
│                                          │
│  选中的文字 → "Python 教程"  ←────────────│
│                                   [🔮] 👈│ ← 悬浮球
│                                          │
└──────────────────────────────────────────┘
```

##### 核心功能

**1. 快捷搜索 + 默认书签**
- 点击悬浮球 → 展开侧边面板
- 显示：搜索引擎快捷列表 + 默认设置的书签
- 点击搜索引擎 → 直接搜索
- 点击书签 → 新标签页打开网站

**2. 选中文本快捷搜索**（🔥 高效功能）
- 在网页中**选中任意文字**
- 点击悬浮球 → 自动获取选中的文本
- 点击搜索引擎 → **直接进入该引擎的搜索结果页**
- 比传统方式快 3 倍：无需复制 → 切换标签 → 粘贴 → 搜索

##### 操作方式

| 操作 | 效果 |
|------|------|
| **左键点击悬浮球** | 展开侧边面板（搜索引擎 + 书签） |
| **Alt + 左键点击** | 打开 FavsHub 侧边栏（完整功能） |
| **网页选中文字 + 点击搜索引擎** | 直接在该搜索引擎中搜索选中的文字 |
| **鼠标离开面板** | 自动收起面板 |

##### 使用场景

**场景 1：快速搜索，无需切换**
```
正在浏览网页 → 需要搜索
→ 点击悬浮球 → 面板展开
→ 显示搜索引擎 + 默认书签
→ 点击搜索或书签，新标签页打开
→ 继续浏览，无缝衔接
```

**场景 2：选中文字直接搜索**（🔥 最高效）
```
阅读文章 → 看到不懂的术语 "Python 装饰器"
→ 选中文字 "Python 装饰器"
→ 点击悬浮球 → 点击 "Google"
→ 直接在 Google 搜索结果页
→ 比传统方式快 3 倍！
```

**场景 3：多引擎对比搜索选中文本**
```
选中文字 "React vs Vue"
→ 点击悬浮球 → 分别点击 "Google"、"Bing"
→ 两个引擎的搜索结果同时打开
→ 对比结果，找到最佳答案
```

### 📱 侧边栏模式

**随时访问书签，不离开当前页面**

#### 什么是侧边栏模式？

侧边栏模式在浏览器右侧打开一个紧凑的书签面板，类似 Edge 的侧边栏：

```
┌────────────────────────────┐
│   当前网页内容              │
│                            │
│   正在浏览网页...           │
│                            │
└────────────────────────────┤
│ 📑 FavsHub 侧边栏          │
├────────────────────────────┤
│ 🔍 [搜索]                  │
│ 📁 书签栏                   │
│  ├ 📂 工具                 │
│  ├ 📂 学习                 │
│ 🎨 书签卡片                 │
│  ┌──────┐ ┌──────┐       │
│  │书签1 │ │书签2 │       │
│  └──────┘ └──────┘       │
└────────────────────────────┘
```

#### 打开侧边栏

**方法 1：快捷键（推荐）**
- Windows/Linux：`Alt + B`
- Mac：`Command + B`

**方法 2：扩展图标**
- 右键浏览器工具栏的 FavsHub 图标
- 选择"打开侧边栏"

#### 侧边栏特点

| 特性 | 说明 |
|------|------|
| 📐 **紧凑布局** | 隐藏页脚、欢迎语、快捷链接，最大化利用空间 |
| 📏 **全宽显示** | 书签容器宽度 100%，充分利用侧边栏宽度 |
| 🔗 **可配置打开方式** | 设置书签点击后：新标签页打开 / 侧边栏内打开 |
| 🔄 **随时切换** | 再次按快捷键关闭/打开 |

#### 配置打开方式

**设置路径：** 设置 → 侧边栏设置

| 选项 | 说明 |
|------|------|
| **新标签页打开** | 点击书签在新标签页打开（推荐） |
| **侧边栏内打开** | 点击书签在侧边栏 iframe 内显示 |

## 常见问题

### Q1: FavsHub 会修改我的书签数据吗？

**A:** 不会。FavsHub 只是 Chrome 书签的"展示器"，所有数据存储在 Chrome 原生书签系统中。您通过 FavsHub 做的任何操作（编辑、删除、排序）都会同步到 Chrome 书签，反之亦然。

---

### Q2: 书签数据保存在哪里？如何备份？

**A:**
- **书签数据**：保存在 Chrome 浏览器内置的书签系统中
- **查看方式**：地址栏输入 `chrome://bookmarks` 查看
- **备份方式**：使用 Chrome 书签管理器导出 HTML 文件

FavsHub **不创建独立数据库**，无需额外备份。

---

### Q3: 书签卡片的颜色是如何生成的？

**A:** 系统自动从网站的 favicon（网站图标）提取主色和次色，经过智能算法生成渐变色背景。

**流程：**
1. 加载网站 favicon
2. 分析图片像素颜色频率
3. 提取最高频的两种颜色
4. 调整明暗度后应用为渐变背景

**为什么有些书签没有颜色？**
- favicon 加载失败（网络问题）
- favicon 为纯色或透明
- 颜色缓存过期

**解决方法：** 刷新页面即可。

---

### Q4: 如何禁用搜索建议？

**A:** 打开设置 → 搜索设置 → 关闭对应开关：
- **书签建议**：关闭后搜索时不显示书签建议
- **历史记录建议**：关闭后搜索时不显示历史建议

---

### Q5: 拖拽排序后为什么顺序又恢复了？

**A:** 拖拽后会立即调用 Chrome API 更新书签顺序，系统每 30 秒同步一次显示。如果顺序恢复，可能是：
- 同步延迟：等待 30 秒即可
- 缓存不一致：刷新页面解决

---

### Q6: 侧边栏模式如何使用？

**A:**
- **快捷键**：`Alt + B`（Windows）或 `Command + B`（Mac）
- **鼠标操作**：右键扩展图标 → 选择"打开侧边栏"
- **再次按快捷键**：关闭侧边栏

---

### Q7: 如何添加自定义搜索引擎？

**A:**
1. 打开设置 → 搜索引擎管理
2. 点击"添加自定义搜索引擎"
3. 填写：
   - **名称**：例如"公司搜索"
   - **URL 模板**：例如 `https://search.example.com?q={query}`
   - `{query}` 会被自动替换为搜索关键词
4. 保存后即可在搜索框上方看到新引擎

---

### Q8: 如何设置默认首页？

**A:**
1. 右键任意文件夹
2. 选择"设为默认首页"
3. 文件夹旁会出现 ⭐ 标记
4. 下次打开新标签页时，自动显示该文件夹内容

**取消方法：** 右键该文件夹 → "取消默认首页"

---

### Q9: 支持同步吗？多设备能用同一套书签？

**A:**
- **书签数据**：通过 Chrome 账号同步，多设备自动同步
- **FavsHub 设置**：部分设置支持 Chrome 同步（跨设备生效）
- **本地设置**：部分设置仅本地生效（如背景主题）

---

### Q10: 如何卸载 FavsHub？

**A:**
1. 打开 `chrome://extensions/`
2. 找到 FavsHub
3. 点击"移除"

**卸载后影响：**
- ✅ 您的 Chrome 书签**不会丢失**
- ✅ 浏览器恢复默认新标签页
- ✅ FavsHub 的设置数据被清除
